How to use in Node.js or Bun

1. Make sure spec-sniffer.js is in the same directory.

2. Create a file named test.js with the following content:

const { getSystemSpecs } = require('./spec-sniffer.js');

async function runTest() {
    console.log("Detecting system specs in Node.js...");
    const specs = await getSystemSpecs();
    console.log(JSON.stringify(specs, null, 2));
}

runTest();

3. Run the script from your terminal: node test.js (or bun test.js)
